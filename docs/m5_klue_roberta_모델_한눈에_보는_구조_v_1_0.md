> **작성 일자**: 2026-05-28

## 1. 비유: 채점하는 국어 선생님

```
┌─────────────────────────────────────┐
│  국어 선생님 (사전학습된 KLUE-RoBERTa) │
│   ↑                                  │
│  한국어 책 수백만 권 읽고 자란 상태    │
│   ↑                                  │
│  채점 5000건 보고 "0~30점 매기는 법" │
│   학습 (3 epoch fine-tune)           │
└─────────────────────────────────────┘
            ↓
   학생 에세이 1편 보여줌
            ↓
   "이 글은 약 24.7점"
```

## 2. 전체 흐름 (3단계)

```
┌──────────────┐    ┌─────────────────┐    ┌──────────┐
│ 학생 에세이   │ →  │ KLUE-RoBERTa     │ →  │ 점수      │
│ (한국어 텍스트)│    │ + 회귀 헤드      │    │ (0~30)   │
└──────────────┘    └─────────────────┘    └──────────┘
   "수 년간                 12층                  24.73
   인터넷을 통한            transformer
   악플과..."
```

## 3. 모델 구조 (계층별)

```
입력: "수 년간 인터넷을 통한 악플과..."
                          │
                          ▼
   ┌──────────────────────────────────┐
   │ Step 1: 토큰화                    │   글자 → 숫자 변환
   │   "수" → 1857                     │   (최대 256개)
   │   "년" → 3492                     │
   │   "간" → 884                      │
   │   ...                             │
   └──────────────────────────────────┘
                          │
                          ▼
   ┌──────────────────────────────────┐
   │ Step 2: 임베딩                    │   각 토큰을 768차원
   │   각 토큰 → 768개 실수 벡터        │   벡터로 변환
   └──────────────────────────────────┘
                          │
                          ▼
   ┌──────────────────────────────────┐
   │ Step 3: RoBERTa 본체 (12층)       │   ← 사전학습된 부분
   │                                   │   한국어 책 수백만 권
   │   각 층마다:                       │   읽고 학습한 지식
   │   • 토큰들이 서로 "보고 듣고"      │
   │   • 문맥 의미 점차 정제            │
   │                                   │
   │   (총 약 68M 파라미터)             │
   └──────────────────────────────────┘
                          │
                          ▼
   ┌──────────────────────────────────┐
   │ Step 4: [CLS] 벡터 추출           │   에세이 전체 의미를
   │   에세이 한 편 → 768차원 요약 벡터  │   하나의 벡터로 요약
   └──────────────────────────────────┘
                          │
                          ▼
   ┌──────────────────────────────────┐
   │ Step 5: 회귀 헤드 (Linear)        │   ← fine-tune으로 새로
   │   768개 → 1개 숫자                │   학습되는 부분
   │   score = w·vector + b           │
   └──────────────────────────────────┘
                          │
                          ▼
   ┌──────────────────────────────────┐
   │ Step 6: 0~30 범위로 자름           │
   │   clip(score, 0, 30)             │
   └──────────────────────────────────┘
                          │
                          ▼
                       24.73
```

## 4. 입력 데이터 (실제 예시)

### 원본 (학생이 쓴 글)

```json
{
  "essay_id": "ESSAY_69855",
  "essay_type": "찬성반대",
  "student_grade": "고등_2학년",
  "paragraph": [{
    "paragraph_txt": "수 년간 인터넷을 통한 악플과 네티켓을
    지키지 않는 사람들 때문에 인터넷 실명제가 다시 사회에
    대두되고 있습니다. 인터넷 실명제란 무엇인지..."
  }]
}
```

### 모델이 보는 것 (토큰 ID)

```
[CLS] 수 년간 인터넷을 통한 악플과 네티켓을 ... [SEP]
  ↓
[2, 1857, 3492, 1124, 2891, 5621, 9043, ..., 3]
   ↑                                          ↑
   문장 시작                              문장 끝
   (최대 256개)
```

### 모델이 **안 보는** 것 (의도적 차단)

| 항목 | 차단 이유 |
|---|---|
| 학생 이름/학교 (없음, 데이터에 PII 없음) | 개인정보 보호 |
| `student_grade` ("고등_2학년") | 메타데이터 누설 방지 |
| 채점자 점수 (rater_1/2/3) | 정답 누설 |
| essay_type ("찬성반대") | 정답 힌트 가능성 |

## 5. 출력 데이터

### 모델 직접 출력 (raw)

```
회귀 헤드 출력: 24.7281...  (단순 실수, 음수나 30 초과도 가능)
```

### 후처리 후 (저장 형식)

| 컬럼 | 값 | 의미 |
|---|---|---|
| essay_id | ESSAY_69855 | 에세이 ID |
| prediction | **24.73** | 모델 점수 (0~30) |
| prediction_rounded | 25 | 정수 반올림 (QWK 계산용) |
| target_essay_scoreT_avg | 25.67 | **실제 정답** (3명 채점자 평균) |
| score_band | high_20_30 | 점수 구간 |

이 결과가 `fold_0_predictions.csv`에 5003개 에세이 저장.

## 6. 손실 함수 (학습 시 사용)

```
         예측 점수            정답 점수
          (24.73)              (25.67)
            │                    │
            └────── 빼기 ───────┘
                     │
                  (-0.94)
                     │
                  제곱
                     │
                  (0.8836)         ← 이 1편의 오차
                     │
        (모든 학습 에세이에 대해)
                     │
                  평균
                     │
                  MSE            ← 최종 손실 값
                  (예: 4.2)
```

수식:
```
MSE = (1/N) × Σ (예측ᵢ - 정답ᵢ)²
```

| 상황 | 손실 | 의미 |
|---|---|---|
| 예측 25, 정답 25 | 0 | 완벽 |
| 예측 25, 정답 26 | 1 | 작은 오차 |
| 예측 25, 정답 28 | 9 | 큰 오차 → **9배 더 강한 학습 신호** |
| 예측 15, 정답 25 | 100 | 매우 큰 오차 → 강력한 학습 |

→ MSE는 **큰 오차를 더 강하게 처벌**. 채점 모델이 극단적 실수(20점차 등)를 줄이도록 학습됨.

## 7. 학습 과정 (한 epoch)

```
   학습 시작
       │
       ▼
   ┌─────────────────────────────────┐
   │ 학습 에세이 5003개 중             │
   │ 16개씩 배치로 나눔                │
   └─────────────────────────────────┘
       │
       ▼
   ┌─────────────────────────────────┐  ← 반복 (5003/16 ≈ 313번)
   │ 1. 배치 16개 forward pass        │
   │ 2. MSE 손실 계산                 │
   │ 3. 손실 역전파 (backprop)        │
   │ 4. 가중치 업데이트 (AdamW)        │
   │    learning_rate=2e-5            │
   └─────────────────────────────────┘
       │
       ▼
   1 epoch 완료
       │
       ▼
   총 3 epoch 반복
       │
       ▼
   검증 데이터로 평가
   (MAE, RMSE, QWK 계산)
```

## 8. 학습되는 가중치 (어디가 바뀌는가)

| 영역 | 파라미터 수 | 변화 |
|---|---|---|
| 임베딩 | ~24M | 미세 조정 |
| RoBERTa 12층 본체 | ~43M | 미세 조정 (lr=2e-5로 작게) |
| Pooler | ~590K | 미세 조정 |
| **회귀 헤드 (768→1)** | **769** | **0에서 시작 → 가장 큰 변화** |

> 사전학습 부분(임베딩 + 12층)은 이미 한국어를 알고 있어서 살짝만 조정. 회귀 헤드는 채점을 처음 배우는 부분이라 가장 많이 변화.

## 9. 입출력 한눈에 요약

| 영역 | 항목 | 값 |
|---|---|---|
| **입력 형식** | 자연어 | 한국어 에세이 본문 (raw text) |
| **입력 텐서** | input_ids | (batch, ≤256) 정수 |
| **입력 텐서** | attention_mask | (batch, ≤256) 0/1 |
| **모델 본체** | KLUE-RoBERTa-small | 68M 파라미터, 12층 |
| **출력 텐서** | logits | (batch, 1) 실수 |
| **출력 후처리** | clip + round | 0~30 정수 또는 실수 |
| **손실 함수** | MSE | `(pred - true)²` 평균 |
| **최적화** | AdamW | lr=2e-5, weight_decay=0.01 |
| **학습량** | 3 epoch | 5003개 에세이 × 3회 |

## 10. 비유로 다시: 학습이 끝난 후

```
국어 선생님이 5000편 채점 연습 마침
              │
              ▼
   새 학생 글 "산업혁명의 영향은..."
              │
              ▼
   읽고 (256 토큰 안에서)
              │
              ▼
   "음, 어휘는 풍부하지만 결론이 약하고
    논리 흐름이 중간에 흐트러지네"
   (← 이 판단은 [CLS] 벡터 768차원에 인코딩)
              │
              ▼
   회귀 헤드: "이 정도면 22.4점"
              │
              ▼
   최종 점수 출력
```

채점 일관성은 학습 데이터(5003편의 인간 채점자 3명 평균)와 같은 분포의 글에서 정확. 분포가 다른 글(예: 대학생 수준 또는 어린이 글)에서는 정확도가 떨어질 수 있음.
